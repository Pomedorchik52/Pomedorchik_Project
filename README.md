# Anitime  
Современный взгляд на сайты для просмотра аниме

## О проекте
**Anitime** — это учебный веб-проект, созданный с целью пересмотреть подход к дизайну и функциональности платформ для просмотра аниме.  
Многие существующие сайты выглядят устаревшими или перегруженными, что делает взаимодействие с ними неудобным.  

Anitime предлагает минималистичный интерфейс, продуманную навигацию и современный визуальный стиль, который помогает сосредоточиться на контенте.

Название проекта отражает тематику и не связано с какими-либо сторонними брендами или компаниями.  
Все интерфейсы, структура и визуальные решения разработаны самостоятельно.
